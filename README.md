## Installation
####1. Clone this project or Download that ZIP file

$ git clone https://github.com/sagarmal624/reactjsNavbarExample.git

####2. Run the following commands

$ npm install
$ npm start

####3. To build
$ npm run build

####4. To clean up old build products
$ npm run clean
